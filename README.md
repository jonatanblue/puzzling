# Puzzlet example

This is a simple example of how to use the Puzzlet API.

## Install

Pre-requisites:
- Node.js
- Typescript
- Yarn

```bash
yarn
```

## Prepare

Set up the environment variables in a `.env` file in the root directory.

```bash
# .env
PUZZLET_API_KEY=""
PUZZLET_APP_ID=""
PUZZLET_PROMPT_TEMPLATE=""
PUZZLET_PROMPT_NAME=""
OPENAI_API_KEY=""
```

## Run

```bash
yarn start
```
